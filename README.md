# Dimes Hazine Adası — Mini-Oyun Geçiş Kapısı (Demo)

Dimes Hazine Adası (CreateJS ada-ilerleme oyunu) içine, her adanın 6. görevinde
"geçiş kapısı" eklenmiş statik demo. Kullanıcı bir sonraki adaya geçmek için ya
bir mini-oyun oynar (başarısız olursa geçemez) ya da görevleri tamamlar.

- Sahte (mock) veriyle çalışır, sunucu gerektirmez — saf statik.
- Alttaki DEMO paneli: istediğin oyunu hemen dene veya gerçek akışı yürü.
- 1:1 eşleme: Milkshake→Paketleme, Smothie→Meyve, Cool Lime→Kalite, Narince→Boru, Obsesso→Çark.

Canlıya taşıma: kanca noktası `DimesHazineAdasi.js` → `LmsUpdateUpLevel(IslandSubLevel===6)`.
